# Langkah-langkah
## Buat koneksi dengan database
- import modules (mongoose, dotenv).
- buat arrow func secara async.
- periksa apakah uri mongodb sudah tersedia di dotenv.
- jika tersedia, koneksikan dan cetak host koneksi.
- jika gagal, cetak error dan exit dari proses.
- export fungsi tersebut untuk digunakan di index.
